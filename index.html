A ROS Extension for ScratchX designed for Turtlebots

Welcome to EasyROS

ROS (Robot Operating System) is a set of software and tools used to create robotics applications. ROS can interact with many programming languages to allow them to control robots that support it. JavaScript is one of these languages, and the EasyROS Scratch Extension will provide an interface for young learners in Computer Science and Computational Thinking. EasyROS can be run on the laptops provided with Turtlebots, or alternatively from another PC on the same network, and the extension provides functionality for connected across the network easily.

Turtlebots

Turtlebots are a low-cost but powerful piece of robotics equipment. They offer a laser sensor(Xbox Kinect), and a bumper sensor. They're extremely portable and a brilliant base for starting to program robots. The robot bases are very robust, making them ideal for teaching children to program robots with Scratch.

Requirements

ScratchX

  • ScratchX works on Google Chrome, tests with Mozille Firefox didn't allow the extension to be loaded.
  • If using Chrome on a linux based machine, be sure to install the official Google Chrome application, and not Chromium, as this also doesn't seem to be compatible with ScratchX.
  • An internet connection is not required. For using ScratchX on the robots' laptops, a HTTPServer can be run, by downloading ScratchX here
  • An internet connection, however, will allow the official ScratchX website to be used to load the extension.

Turtlebot Laptops

  • Some Knowledge of the Ubuntu Operating System is helpful, although not integral.
  • However, knowledge of the Ubuntu Operating System is expected for the initial setup of the Turtlebots.
  • This project has been designed to be compatible with Ubuntu 14.04, running ROS-Indigo.
  • A full installation guide for the Turtlebots, including the ROS-Indigo installation can be found here
  • This installation should be inclusive of all of the drivers used in this extension.
  • An internet connection is required to install this.

Turtlebot Setup

Turtlebot Setup (Required for both)

  • These steps assume you have followed the guide to install ROS-Indigo linked above and have a successful installation of ROS-Indigo and the Turtlebot proprietary drivers.
  • Download the .launch file that can be found here
  • This can be anywhere on the machine, as long as you're aware of its location.
  • To get the Turtlebot ready for the extension you need to run this launch file. This can be done by running the following on a terminal, whilst in the directory of the launch file. roslaunch scratch_turtlebot.launch
  • This should start up all of the drivers required for the extension.

Using ScratchX on the Ubuntu Laptops

  • These steps are to be followed if you're not using additional computers to connect to the Turtlebot laptops.
  • If you do not have an internet connection whilst using the extension, you should download the ScratchX files listed above, and from the base directory of the folder downloaded run the following. python -m SimpleHTTPServer
  • If you do have an internet connection you can instead use the online version here
  • When loading the extension it will ask for an IP address. Use 'localhost'. ### Using windows PCs to connect to the Turtlebots
  • If you're using different PCs to connect to the turtlebots, they need to be on the same network.
  • The connection should not use a proxy.
  • Load ScratchX on the PC that you want Scratch loaded on, and go here
  • It will ask you for an IP Address. To get this, open a terminal on the Turtlebot's laptop and type ifconfig and hit enter.
  • If you're using wifi it will probably come up under the name wlan0, find the number after inet addr in a similar format to 192.168.1.2
  • This is the IP address to use to connect to this specific Turtlebot.
